# User Framework

Use the user's personal investing system as the default scaffold:

1. Long-term trend logic
2. National planning and policy-study logic
3. Real-world relationship and transmission understanding
4. Medium-term industry logic
5. Short-term execution optimization

For this skill, the priority is:

- First: strengthen Layer 1 long-term macro understanding
- Second: strengthen Layer 2 medium-term industry logic
- Third: only translate into Layer 3 execution if the user explicitly asks

Default interpretation rule:

- If the user asks "怎么看", answer first from long-cycle regime and policy structure.
- If the user asks "哪些方向受益", answer second from medium-term industry transmission.
- If the user asks "怎么操作", give only a light execution translation unless they ask for deeper trading detail.
