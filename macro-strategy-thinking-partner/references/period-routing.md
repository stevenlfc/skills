# Period Routing

This skill must separate the macro corpus into two periods.

## Period A: 924之前

Definition:

- `文件 / 来自：分享 / 司总往期 / 宏观策略`
- Exclude the `9.24之后` subfolder when answering this period by default.

Interpretation:

- This is the earlier regime.
- Use it when the user asks about pre-924 views, older macro logic, or wants to understand the earlier narrative state before the later policy / market shift.

## Period B: 924之后（含2026）

Definition:

- `文件 / 来自：分享 / 司总往期 / 宏观策略 / 9.24之后`
- `文件 / 来自：分享 / 司总往期 / 2026 / 宏观策略`

Interpretation:

- Treat these two folders as the same later regime bucket.
- Use this bucket when the user asks about the post-924 phase, 2025-2026 macro views, or the later-stage market narrative after the regime changed.

Observed examples in this bucket include:

- `【首席圈儿.宏观】2025年2月28日 近期市场观点_文稿.pdf`
- `【首席圈儿.宏观】2025年5月16日 二季度市场观点&超额收益的一些基本策略_文稿.pdf`
- `【首席圈儿.宏观】2025年7月18日 市场展望：会是2014、2015年吗？_文稿.pdf`
- `【首席圈儿.宏观】2025年9月5日 站在3800之上_文稿.pdf`

## Ambiguity Rule

If the user asks a broad question like:

- `怎么看接下来市场？`
- `现在科技怎么配？`
- `宏观上怎么理解？`

Ask first:

`你问的是924之前，还是924之后（含2026）？`

Only skip that clarification if the user's wording already pins the period down.

## Comparison Rule

If the user explicitly wants comparison:

1. Summarize Period A thesis
2. Summarize Period B thesis
3. Explain what changed:
   - policy stance
   - liquidity condition
   - market confidence
   - industry leadership
   - allocation implications
